# ⚖️ Calculadora de IMC Moderna

Este é um projeto de uma calculadora de Índice de Massa Corporal (IMC) desenvolvido com um design moderno, responsivo e interativo. 

O projeto foi criado para praticar conceitos de manipulação do DOM com JavaScript, estilização avançada com CSS e estruturação semântica com HTML.

---

## 🚀 Funcionalidades

- **Cálculo de IMC:** Retorna o valor exato do IMC com base no peso e altura informados.
- **Classificação Dinâmica:** Exibe uma mensagem personalizada (Peso Ideal, Sobrepeso, Obesidade, etc).
- **Feedback Visual:** As cores do resultado mudam conforme a categoria do IMC (Verde, Amarelo ou Vermelho).
- **Totalmente Responsivo:** Design adaptado para telas de computadores, tablets e celulares.
- **Interface Limpa:** Estilo Dark Mode com foco em UX (Experiência do Usuário).
